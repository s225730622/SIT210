# Task 2.1P - Sending Temperature and Light Data to the Web #

### TASK DESCRIPTION ###
Create a system which records temperature and light levels.

This task introduces the concepts of collecting analogue and digital sensor data and sending data from an Arduino device to the web.
<br/>

### SOLUTION CODE EXPLANATION ###
* I have included all required libraries and declared all variables which will be referenced throughout the code
* I have utilized modular programming by separating reading Temperature levels, reading Light levels, and collecting this data into respective functions
* The program then pushes the data to ThingSpeak
