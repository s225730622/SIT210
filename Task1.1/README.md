# Task 1.1P - Switching ON Lights: Modular Programming Approach #


### TASK DESCRIPTION ###
Design a system by writing a program which switches ON the porch light and the hallway light when a switch/push button is pressed.

The porch light should stay ON for 30 seconds and the hallway light should stay ON for 60 seconds.  
<br/>

### SOLUTION CODE EXPLANATION ###
* I have declared three variables at the start of the program before the setup function with initial values.\
* These variables have are assigned a mode and set accordingly within the setup function.\
* The loop function consists of an If statement which states:
  * If the button is pushed from LOW (which is off) to HIGH (which is on), then compute the code as follows:
      1. Turn the porch light on for 30 seconds, then turn off
      2. Turn the hallway light on for 60 seconds, then turn off
